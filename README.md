# SOC Incident Reports

This repository contains structured incident investigation reports created as part of hands-on SOC training and lab exercises.

## Focus Areas
- Log analysis (Web, Auth, Application)
- Web attack detection (IDOR, XSS, SQLi)
- Incident triage and classification
- Evidence-based reporting
- SOC L1/L2 documentation workflows

## Tools & Platforms
- SIEM concepts (Splunk-style analysis)
- LetsDefend
- TryHackMe
- Obsidian (Markdown-based reporting)
- Git version control

## Note
All reports are based on simulated lab environments and training scenarios.
